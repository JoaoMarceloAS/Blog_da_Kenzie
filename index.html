CORES EM HEXADECIMAL CSS: PASSO A PASSO PARA APLICAR CORES NO HTML

Em uma página web, normalmente utilizamos propriedades de documentos em formato CSS (Cascade Style Sheet) para personalizar a aplicação, essas propriedades são das mais diversas. Um dos valores de algumas dessas propriedades como background-color, color e etc., é definir uma cor, para isso, temos várias formas de expressar o valor da cor, podemos defini-las pelo nome da cor, rgb passando os valores como decimais, HSA, HSL ou com código hexadecimal.
